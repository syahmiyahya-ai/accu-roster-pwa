Install ACCU Roster

Add the roster app to your phone Home Screen.

Open ACCU Roster PWA Open poster image

iPhone / iPad

  1. Open Safari.
  2. Open the ACCU Roster PWA link.
  3. Tap Share.
  4. Tap Add to Home Screen.
  5. Name it ACCU Roster and tap Add.

Android

  1. Open Chrome.
  2. Open the ACCU Roster PWA link.
  3. Tap the three-dot menu.
  4. Tap Install app or Add to Home screen.
  5. Name it ACCU Roster and tap Install/Add.

If you see 403 or the old app, delete the old Home Screen icon first. Then install again from the v12 link.
